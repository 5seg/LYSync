# LYSync

YouTubeのプレイリストで、追加差分が発生した場合にそれらをダウンロードします。

`yt-dlp`を何度も実行するよりも、実行時間と無駄なリクエストを削減できます。

## 使用前にお読みください

- LYSyncはあらかじめダウンロードされたプレイリストの変更を検知し、追加分をダウンロードすることを目的としています。まだプレイリストをローカルにダウンロードしていない場合は、各自で行ってください。

> (ダウンロードが面倒であれば、空の`urls.txt`を作成してください。)

- デフォルトでは動画ではなく`mp3`ファイルとしてダウンロードを行います。([引数](#ダウンロード時に使用する引数))

## 挙動

1. `PLAYLIST_URL`に設定されたプレイリストを取得します。
2. `urls.txt`(または`URLS_FILE`で定義されたファイルの名称)が存在しない場合、新規作成 & 書き込みをして**終了**します。そうでなければ次のステップに進みます。
3. 差分を確認し、追加がある場合はそれらをダウンロードします。削除されたURLは無視されます。

## ダウンロード時に使用する引数

ダウンロード時に、このスクリプトは`yt-dlp`に次の引数を渡します:

`-t mp3 --embed-metadata --embed-thumbnail --retries infinite --fragment-retries infinite --retry-sleep fragment:exp=1:120 --sleep-interval 10 -P "$SAVE_DIR"`

必要に応じて引数を編集してください。`sync.sh`の23行目に存在します。
